澤井佑樹のプロフィール紹介

January 24, 2023

澤井佑樹と言います。 企業の研究所でコンピュータサイエンスに関する研究に従事しています。 今まで、組合せ最適化、機械学習、高機能暗号を対象に研究してきています。 現在は株式会社デンソーにて働きながら、社会人博士として電気通信大学 岩本・渡邉研究室にて博士号取得を目指しています。

出身

出身地は滋賀県の湖北地方です。 水と自然豊かで、暮らしやすい場所です。 現在は故郷を離れて生活していますが、 ゆくゆくは故郷発展のために貢献できたらと考えています。

所属

  • 株式会社デンソー AI研究部 基盤技術研究室 研究企画推進課
  • 電気通信大学 大学院情報理工学研究科 情報学専攻 博士課程後期D1

研究分野

  • 高機能暗号
    • 特に、ゼロ知識証明（zk-SNARKの効率化）
  • 組合せ最適化
    • スケジューリング問題
  • 機械学習
    • 乗員の眠気推定、故障検知
