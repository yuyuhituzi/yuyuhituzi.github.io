自己紹介

私は高機能暗号と呼ばれる分野の中でも特に、自分の持つ知識を明かさずに知識を持つことを相手に証明する「ゼロ知識証明」の効率化を目指す研究をしています。ゼロ知識証明を応用することで、

  • 仮想通貨における具体的な取引内容（送金額など）は明かさずに、正しく取引が成立したことを証明できる
  • ユーザはプライベートな情報（年齢やパスワードなど）を明かさずに、サービス認証ができ、サービスを享受できる

など、プライバシ保護とデータ利活用を両立することが可能となります。 そのため、様々なデータを安心して連携させデータ流通を促進することを目指すSociety5.0などへ貢献できる重要な技術と考えています。
